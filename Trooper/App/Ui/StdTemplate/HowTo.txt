HOW TO USE THE STANDARD THEME TEMPLATE IN YOUR PROJECT
------------------------------------------------
Note: this project is only useful if your using an MVC based web project and you
are using Bootstrap.

1. Include the Trooper project into your solution

2. Add this into the Application_Start method of your Global.asax

   StdTheming.Init();

3. Create common Layout Model and View that all your models and views can inherit.
   Lets call them YourAppModel.cs and YourAppView.cshtml

   YourAppModel.cs
   ------------------------------
   public class YourAppModel : StdLayoutModel
   {
		public int YourSpecificProperty { get; set; }
   }
   ------------------------------

   YourAppView.cshtml
   ------------------------------
   @using Trooper.App.Ui.StdTemplate
   @model YourAppModel

   @{
       Layout = StdTemplate.GetLayoutViewPath();  
   }

   @StdTemplate.RedefineSection(this, "Header")

   @StdTemplate.RedefineSection(this, "LeftNavigation")

   @Stdemplate.RedefineSection(this, "TopSearch")

   @StdTemplate.RedefineSection(this, "MainArea")

   @StdTemplate.RedefineSection(this, "LeftNavigationButtons")

   @StdTemplate.RedefineSection(this, "MainNavigationButtons")
   ------------------------------

4. Your Done!
